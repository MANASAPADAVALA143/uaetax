# 🎬 First Demo Script - 3 Minutes

## Opening (10 seconds)
"Hi, I'm [Your Name]. I built GulfTax AI to solve UAE VAT compliance for CA firms. Let me show you how it works in 3 minutes."

## 1. Dashboard Overview (15 seconds)
- Show the dashboard
- "This is the main overview — you can see VAT due, transactions classified, and filing deadlines at a glance."

## 2. Upload & Classify (60 seconds)
- Go to VAT Classifier
- "I'll upload a CSV with 50 transactions from Q1 2025"
- Drag and drop `test_transactions.csv`
- Click "Classify Transactions"
- "The AI is now classifying each transaction using UAE VAT rules"
- Show results table scrolling
- **Key point:** "Notice the reverse charge transactions — Microsoft Azure, AWS, Google. These are the most complex cases, and the AI handles them automatically."
- "All 50 transactions classified in seconds with confidence scores."

## 3. Generate VAT Return (50 seconds)
- Go to VAT Return page
- "Now I'll generate the Q1 2025 VAT return"
- Select Q1 2025, click "Generate Return"
- "The system calculated all 8 FTA boxes automatically"
- Scroll through boxes 1-7
- **Highlight Box 8:** "Box 8 shows AED -2,069.92 refundable. This matches our expected calculation exactly."
- "This would normally take hours — we did it in 3 minutes."

## 4. Reconciliation (30 seconds)
- Go to Reconciliation
- "Before filing, we run reconciliation"
- Select return, click "Run Reconciliation"
- "It compares invoices against the return and flags any mismatches"
- Show results: "Matched" or mismatch details
- "This catches errors before you file with FTA."

## 5. Download Reports (15 seconds)
- Back to VAT Return
- "You can download the return as PDF — FTA-ready format"
- "Or Excel with 5 detailed sheets for your records"

## Closing (10 seconds)
"That's it — 3 minutes from upload to ready-to-file VAT return. Questions?"
